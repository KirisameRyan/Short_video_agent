# 项目概览

## 🎯 项目目标

构建一个完整的抖音短剧内容分析和创作辅助工具链，从下载到分析生成创作指南。

## 🔄 工作流架构

```
┌─────────────────┐
│  抖音视频链接    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  批量下载视频    │ ← new.py
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  提取字幕/转录   │ ← extract_douyin_transcripts.py
│  (Whisper/FunASR) │    或 FunASR 工具链
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  质量评分        │ ← score_dialogues_deepseek.py
│  (S/A/B/C级)    │    DeepSeek API
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  内容优化        │ ← dialogue_enhancer.py
│  (仅S/A级)      │    DeepSeek API
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  生成创作方法论  │ ← batch_process_docs.py
│  (结构化分析)   │    DeepSeek API
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  上传知识库      │ ← upload.py
│  (Dify)         │    Dify API
└─────────────────┘
```

## 📊 核心组件

### 1. 下载模块

- **douyin-downloader**：基于 GitHub 开源项目
- **new.py**：自定义批量下载脚本
  - 从 `urls.txt` 读取链接
  - 逐个调用下载器
  - 错误处理和重试机制

### 2. 转录模块

#### 方案 A：Whisper

- **extract_douyin_transcripts.py**
- 优点：准确度高，支持多语言
- 缺点：速度慢，需要 GPU 加速

#### 方案 B：FunASR

- **step1_collect_files.py**：收集输出
- **check_format.py**：格式检查
- **collect_transcripts.py**：转换为台词本
- 优点：速度快，中文识别好
- 缺点：需要单独安装阿里工具

### 3. 质量评估模块

- **score_dialogues_deepseek.py**
- 使用 DeepSeek API（通过硅基流动）
- 评分标准：
  - **S 级**：剧情紧凑，冲突强烈，情感饱满
  - **A 级**：逻辑清晰，有亮点
  - **B 级**：基本流畅，有改进空间
  - **C 级**：结构松散，质量较低

### 4. 内容优化模块

- **dialogue_enhancer.py**
- 仅处理 S 级和 A 级内容
- 优化方向：
  - 提取核心冲突
  - 标注情感高潮
  - 总结叙事技巧
  - 添加创作启示

### 5. 方法论生成模块

- **batch_process_docs.py**
- 分析所有优化后的文档
- 输出：
  - **高频关键词**（前 30）
  - **冲突类型统计**
  - **情感模式分析**
  - **场景结构**
  - **创作技巧总结**

### 6. 知识库模块

- **upload.py**
- 上传到 Dify 知识库
- 特性：
  - 断点续传
  - 慢速模式（避免速率限制）
  - 自动分块（大文件）
  - 进度保存

### 7. 辅助工具

- **format_json.py**：JSON 格式化（可能用于调试）
- **batch_process_docs.py**：批量文档处理

## 🔑 关键技术

### API 集成

1. **DeepSeek API**（via SiliconFlow）
   - 评分、优化、分析
   - 端点：`https://api.siliconflow.cn/v1`

2. **Dify API**
   - 知识库上传
   - 端点：`http://www.azureflame.cloud/v1`

### 数据处理

- 文本提取和清洗
- 结构化数据生成（JSON）
- 高频词统计
- 情感标签识别

### 稳定性设计

- 断点续传机制

- 进度持久化
- 速率限制控制

## 📊 性能指标

**测试环境**：
- CPU: Intel i7 / Ryzen 7
- GPU: RTX 3060 (Whisper)
- 网络: 100Mbps

**典型处理速度**（100 个视频）：

| 步骤 | 耗时 | 备注 |
|------|------|------|
| 下载 | 1-3h | 取决于网速 |
| 转录 (Whisper) | 2-5h | GPU 加速 |
| 转录 (FunASR) | 30min-1h | CPU 即可 |
| 评分 | 30min | API 调用 |
| 优化 | 20min | 仅处理 S/A 级 |
| 生成方法论 | 10min | 批量分析 |
| 上传 Dify | 30min | 慢速模式 |
